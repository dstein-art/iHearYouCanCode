Loading & Playing Sounds

iHearYouCanCode — by David Stein
Load audio files, play them on click, and control volume and speed
↗ SoundFile Docs
Haven't set things up yet? Do Getting Started first — it shows you how to add p5.js and p5.sound.

1. Load a sound file

A p5.SoundFile holds audio loaded from a file — like an MP3 or WAV. Loading takes time, so use preload() to fetch the file before setup() runs. Call loadSound() with a path or URL. Click the canvas to play it.

The examples below use a short sample hosted online. When you build your own sketch, put audio files in your project folder and use a path like 'sounds/beat.mp3'.

Loading takes time

Audio files don't appear instantly. loadSound() fetches the file in the background. Use isLoaded() to check if it's ready before you call play(). If you try to play too early, nothing happens.

2. Control volume and stop playback

Use amp() to set how loud the sound is — from 0 (silent) to 1 (full volume). Move the mouse up and down while the sound plays. Click to start; click again to stop() it. isPlaying() tells you if the sound is going right now.

3. Loop a sound

A loop() repeats a sound over and over — great for beats and background tracks. Click to start looping; click again to stop(). isLooping() tells you if the sound is set to loop.

4. Change playback speed

rate() changes how fast the sound plays. At 1 it sounds normal. Higher than 1 is faster (and higher pitch). Lower than 1 is slower (and lower pitch). Values below 0 play the sound backwards. Move the mouse left and right to try different rates while the loop plays.

5. Use ProControls for play and volume

Swap click-to-play for on-screen controls. An IconButton toggles play and stop. A MultiSlider sets volume. Call userStartAudio() when the player touches a control, and call proControlBackground() in draw().

Handy SoundFile methods

  • play() — play once from the start
  • loop() — play on repeat
  • pause() / play() — pause and resume from the same spot
  • stop() — stop and reset to the beginning
  • duration() — length of the file in seconds
  • currentTime() — where the playhead is right now
  • jump(time) — jump to a position in the file
  • setLoop(true/false) — turn looping on or off

When you host your own files, you can pass several formats in an array — loadSound(['beat.mp3', 'beat.ogg']) — and the browser picks one that works. See the p5.SoundFile reference.
